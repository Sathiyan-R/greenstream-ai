

# GreenStream AI — Premium SaaS Landing Page

A visually stunning, dark-mode landing page with a **Tesla × OpenAI × Palantir** aesthetic. Neon green + electric blue highlights, glassmorphism, smooth animations, and a data-stream visual language throughout.

---

## 1. Hero Section (Full-screen, cinematic)
- Dark gradient background (black → deep green) with animated floating particles and glowing grid lines
- Bold headline: *"AI That Thinks in Real-Time. Cities That Act in Real-Time."*
- Subheadline describing GreenStream AI's capabilities
- Two CTA buttons: **View Live Demo** and **Explore AI Architecture**
- Smooth fade-in and scale entrance animations

## 2. The Problem Section
- Title: *"The Invisible Energy Crisis"*
- Four animated stat cards with counting-up number animations (30% waste, delayed decisions, HVAC/traffic spikes, underutilized renewables)
- Dramatic reveal on scroll

## 3. The Solution — Animated Pipeline
- Title: *"Meet GreenStream AI"*
- Horizontal animated pipeline: IoT Sensors → Pathway Connectors → Streaming Engine → ML Anomaly Detection → Carbon Intelligence → LLM Assistant
- Each stage glows on hover with animated flowing arrows between nodes

## 4. Core Features — Glass Cards
- Six glassmorphism cards with neon glow hover effects:
  - Live Streaming Intelligence
  - Rolling Window Analytics
  - Carbon Footprint Calculator
  - AI Anomaly Detection
  - RAG-Powered AI Assistant
  - Predictive Load Forecasting
- Cards animate in on scroll with staggered timing

## 5. Live Dashboard Preview
- Realistic mock analytics dashboard showing:
  - Carbon emission gauge
  - Energy usage line chart (using Recharts)
  - Renewable production bar graph
  - Traffic heatmap visualization
  - Anomaly alert notification popup
- Styled as an enterprise SaaS dashboard embed

## 6. AI Assistant Demo
- Mock chat interface with a realistic conversation:
  - User asks about emission spikes
  - AI responds with contextual, explainable insights
- Typing animation effect for the AI response
- Glassmorphism chat bubble styling

## 7. Enterprise Architecture Section
- Clean architecture diagram with glowing connected nodes: IoT → Pathway → Transformations → Vector Index → LLM xPack → Dashboard & APIs
- Animated connecting lines showing data flow

## 8. Footer
- Minimal footer with project branding, hackathon credits, and social/demo links

---

### Design System
- **Colors:** Dark background, neon green (#00FF88) and electric blue (#00D4FF) accents
- **Style:** Glassmorphism panels, subtle glow effects, smooth scroll-triggered animations
- **Typography:** Clean, modern, large bold headings with light body text
